# IT Support Ticket Lab

This repository contains ticket-based IT support practice scenarios focused on Service Desk and End User Support workflows.

The purpose of this lab is to practise how to receive, document, troubleshoot, resolve, and escalate common IT support issues in a structured and business-aware way.

## Focus Areas

* Microsoft 365 user support
* Windows endpoint troubleshooting
* iOS and mobile device support
* Account and access administration
* New starter onboarding and offboarding
* IT asset lifecycle tracking
* Retail/POS system support scenarios
* Incident documentation and escalation
* Knowledge base article writing

## Support Methodology

Each scenario follows a consistent support process:

1. Clarify the user issue and business impact
2. Identify the affected user, device, location, and service
3. Determine whether the issue is user-specific, device-specific, network-related, application-related, or system-wide
4. Perform basic troubleshooting steps
5. Document findings clearly
6. Provide a workaround where possible
7. Escalate with complete notes when required
8. Create or update a knowledge base article if the issue may recur

## Ticket Categories

* Microsoft 365
* Windows
* Network
* iOS / Mobile Device

* Hardware
* Retail / POS
* Onboarding / Offboarding

## Goal

The goal of this repository is to build practical IT support habits: clear communication, structured troubleshooting, accurate documentation, prioritisation based on business impact, and responsible escalation.
