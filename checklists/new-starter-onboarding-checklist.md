# New Starter Onboarding Checklist

## User Details

- Full name:
- Job title:
- Department:
- Manager:
- Start date:
- Location:

## Account Setup

- Microsoft 365 account created
- Email address created
- Required groups assigned
- Shared mailbox access confirmed, if required
- Teams access confirmed
- MFA setup required
- Temporary password process followed

## Device Setup

- Laptop assigned
- Charger assigned
- iPhone/iPad assigned, if required
- Device name recorded
- Serial number recorded
- Asset tag recorded
- Required software installed
- Windows/macOS updates checked
- Security tools confirmed

## Business Applications

- Retail/POS access, if required
- Inventory system access, if required
- File share access
- Department-specific applications

## Handover

- Login tested
- Email tested
- Teams tested
- OneDrive tested
- User provided with basic instructions
- Asset register updated
- Ticket closed with notes
