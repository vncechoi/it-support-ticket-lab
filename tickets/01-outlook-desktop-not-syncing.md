# Outlook Desktop Not Syncing

## Ticket Summary

**User:** Head Office User  
**Department / Location:** Sydney Head Office  
**Device / System:** Windows laptop / Microsoft Outlook  
**Category:** Microsoft 365 / Email  
**Priority:** Medium  
**Status:** Resolved with workaround / Escalated if issue persists  

## User Reported Issue

User reported that Outlook desktop was not receiving new emails.

## Business Impact

Single user affected. Email access is important for daily communication, but Outlook Web is available as a temporary workaround.

## Initial Questions Asked

- When did the issue start?
- Are you able to access Outlook Web?
- Are Teams and OneDrive working?
- Can you send emails, receive emails, or neither?
- Are there any error messages?
- Did you recently change your password?
- Is the issue happening on mobile as well?

## Troubleshooting Steps

1. Confirmed internet connection was working.
2. Confirmed Microsoft Teams was working.
3. Tested Outlook Web access.
4. Restarted Outlook desktop.
5. Checked whether the user was signed into the correct account.
6. Asked whether password or MFA had recently changed.
7. Confirmed whether the issue was isolated to Outlook desktop.

## Findings

Outlook Web was accessible and Teams was working, so the issue appeared to be isolated to the Outlook desktop application rather than a full Microsoft 365 account outage.

## Resolution or Workaround

Provided Outlook Web as a temporary workaround. Recommended further review of Outlook desktop profile, cached credentials, or application update if the issue continued.

## Escalation Notes

Escalate to senior IT or Microsoft 365 administrator if:
- Outlook desktop continues failing after restart
- Profile rebuild is required
- Cached credentials need clearing
- Account/licensing issue is suspected

Escalation should include:
- User affected
- Device name
- Time issue started
- Confirmation that Outlook Web works
- Confirmation that Teams/OneDrive work
- Screenshots of any error messages

## Closure Note

User was informed that Outlook Web could be used temporarily. Issue documented and prepared for escalation if the desktop application problem continued.
